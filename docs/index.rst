Django-Skipjack documentation
=============================

`Skipjack`_ is a payment gateway. Django-Skipjack is MIT-licensed and makes it
easy to use Skipjack as the payment gateway for your `Django`_ application.

Includes the AuthorizeAPI implementation for payment submission useful in
shopping cart integration and integration into custom CRM payment solutions.
